Hóa học: Nước: H2O; Axit Sulfuric: H2SO4

Các công thức hóa học phổ biến:

  • Natri hydroxit: NaOH
  • Axit clohidric: HCl
  • Canxi cacbonat: CaCO3
  • Natri Clorua: NaCl

Phản ứng hóa học:

H2SO4 + 2NaOH → Na2SO4 + 2H2O

CaCO3 + 2HCl → CaCl2 + H2O + CO2↑

Toán học: Phương trình bậc 2: ax2 + bx + c = 0

Công thức nghiệm phương trình bậc 2:

x = (-b ± √(b2 - 4ac)) / 2a

Ví dụ:

Giải phương trình: 2x2 + 5x - 3 = 0

a = 2, b = 5, c = -3

Δ = b2 - 4ac = 52 - 4×2×(-3) = 25 + 24 = 49

x1 = (-5 + √49) / (2×2) = (-5 + 7) / 4 = 0.5

x2 = (-5 - √49) / (2×2) = (-5 - 7) / 4 = -3

Đặc biệt:

- Nếu Δ > 0: Phương trình có 2 nghiệm phân biệt

- Nếu Δ = 0: Phương trình có nghiệm kép x = -b/(2a)

- Nếu Δ < 0: Phương trình vô nghiệm

Tin học: Vòng lặp For 
For i = 0 To 10
    Tong = Tong + i 
Next

Các ví dụ với ngôn ngữ khác:

JavaScript:

let tong = 0;
for (let i = 0; i <= 10; i++) {
    tong += i;
}
console.log(tong); // Kết quả: 55

Python:

tong = 0
for i in range(11):
    tong += i
print(tong)  # Kết quả: 55

C++:

int tong = 0;
for (int i = 0; i <= 10; i++) {
    tong += i;
}
cout << tong; // Kết quả: 55

Công thức tính tổng của n số tự nhiên đầu tiên:

Tổng = n(n+1)/2

Với n = 10: Tổng = 10(10+1)/2 = 10×11/2 = 55
